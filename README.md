# tp4
tp4 gta431 - Castonguay, Jacques, Lallier
Fait sur Python 3.7, avec Anaconda Matplotlib(as plt) et Pandas(as pd).
Le fichier de données csv a été modifié avec des refactors pour enlever certaines lettres causant problème(à,é,è, etc.).
La version sur github est bien la version modifiée.
Le fichier original se trouve ici : http://donnees.ville.montreal.qc.ca/dataset/actes-criminels/resource/c6f482bf-bf0f-4960-8b2f-9982c211addd
Afin de télécharger et exécuter le code, il faut télécharger ce dossier en format .zip et l'extraire dans votre environnement Python. Il faut s'assurer d'avoir Anaconda et les modules Pandas, Jupyter Notebook et Matplotlib pour son bon fonctionnement.
